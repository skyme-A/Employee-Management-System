# Employee Management System

A modern Employee Management System built using React, Vite, and Tailwind CSS. This application helps manage employee records efficiently through a clean and responsive user interface.

## 🚀 Features

* Add new employees
* View employee list
* Edit employee details
* Delete employee records
* Responsive design for desktop and mobile
* Fast development and build performance using Vite

## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS
* JavaScript

## 📂 Project Structure

Employee-Management-System
└── ems
    ├── public
    ├── src
    ├── index.html
    ├── package.json
    ├── vite.config.js

## ▶️ Run Locally

```bash
npm install
npm run dev
```
